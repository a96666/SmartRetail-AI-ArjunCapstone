# smart-retail-ai

Project scaffold for Smart Retail AI.

Structure:

- app/: application code (api, agents, ml, rag, database, services, utils)
- data/: datasets and raw data
- models/: trained model artifacts
- notebooks/: exploratory notebooks
- dashboard/: dashboard/frontend for monitoring
- tests/: unit and integration tests
- frontend/: frontend app (React/Vue/etc)

Run:

```bash
python main.py
```
